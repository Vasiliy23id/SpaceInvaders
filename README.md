# pythonProject
University project like space invaders light clone

The game run from alien_invation.py

Control

Start the game on the left button

    1)Movement - arrows:
        UP
        DOWN
        LEFT
        RIGHT
    2)Fire - SPACE
    3)Exit to menu - Esc
    4)on/off ost music F1
    5)change ost volume - F2(less) F3(more)
